# LWC Chat App powered by Platform Event

Salesforce上で簡易的にユーザー間チャットを実現するLWCコンポーネントです。
プラットフォームイベントをPub/Subしています。
利用される際は、利用するユーザーのプロファイル設定で、LWC_Chat__eオブジェクトのRead/Create権限を付与してから利用してください。
